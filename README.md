# RoRejoin
RoRejoin watches multiple Roblox accounts simultaneously. When one crashes, it auto-rejoins into a fresh server after a configurable delay. Set different places and delays per account, arm periodic auto-kill timers, and receive Discord webhook notifications on every crash, rejoin, and session event.
